# 🚀 Profile Card — Frontend Wizards Stage 1B

A clean, accessible, and responsive **Profile Card** built with **HTML, CSS, and Vanilla JavaScript**.

This project was created for the **Frontend Wizards Stage 1B Task** and follows all required specifications, including semantic HTML, accessibility best practices, responsiveness, and stable `data-testid` attributes for automated testing.

## 📂 GitHub Repository

## ✨ Features

- ✅ Semantic HTML structure
- ✅ Accessible UI with meaningful `alt` text
- ✅ Responsive layout for mobile, tablet, and desktop
- ✅ Live current time display (updates every second)
- ✅ Keyboard-focusable social links
- ✅ Visible focus styles for accessibility
- ✅ Clean modern UI
- ✅ Stable `data-testid` attributes for automated tests
- ✅ Social links open in a new tab securely
